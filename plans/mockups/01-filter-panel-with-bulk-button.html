About Bulk Generation

Bulk generation creates descriptions for multiple images at once.

  • Only images without descriptions will be processed
  • Processing time: ~1 second per image
  • You can continue browsing while processing
  • Progress updates appear in bottom-right corner
  • Maximum 100 images per batch

⚠️ Large batches may take several minutes. Make sure your image-to-text service is running.

Design Notes

  • Location: Bottom of filter panel (Option 1 - Recommended)
  • Count Logic: Calculated on FULL result set (before pagination), only images without embeddings
  • Help Text: Collapsible details element, explains feature without cluttering UI
  • Model Display: Shows current model name so user knows what will be used
  • Empty State: Disabled button + friendly message when 0 images match
  • Filter Lock: When bulk operation active, filters and bulk button disabled with warning banner
  • Large Batch Warning: Shows alert when 100+ images selected, explains 100-image limit
  • Confirmation: Native browser confirm() dialog on click (simple, accessible)
