Design Notes

  • Position: Fixed bottom-right (bottom-4 right-4), floats above all content
  • Persistence: Uses localStorage to survive page reloads, restores state on connect
  • Polling: Exponential backoff (starts 2s, max 10s), resets to 2s on progress
  • Real-time Updates: Listens to ActionCable broadcasts for instant status changes
  • States: 6 distinct states (initial, progressing, complete, errors, connection lost, minimized)
  • Auto-dismiss: 5 seconds after 100% success, stays visible if errors occurred
  • Minimized Mode: Compact badge with circular progress indicator, click to expand
  • Error Handling: Shows connection errors after 3 failed polls, continues retrying
  • Cancel: Sends cancel request to backend, removes in_queue jobs, stops polling
  • Retry: Button appears when failed_count > 0, resubmits failed images only
  • Accessibility: ARIA live regions for screen readers, keyboard shortcut (Escape) to close
  • Time Estimates: Shows estimated time remaining based on (total - completed) * 1 second per image
